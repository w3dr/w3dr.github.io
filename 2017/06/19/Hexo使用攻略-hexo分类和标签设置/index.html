Hexo使用攻略 hexo分类和标签设置

Veröffentlicht am 2017-06-19 | in Hexo

为了方便归类和查看,介绍一下hexo里分类和标签的设置.

添加标签(tags)

$ hexo new page tags
确认站点配置文件里有tag_dir: tags
确认主题配置文件里有tags: /tags
在编辑站点的source/tags/index.md，添加

title: tags  
date: 2015-10-20 06:49:50  
type: "tags"  
comments: false

添加分类(categories)

$ hexo new page categories
确认站点配置文件里有category_dir: categories
确认主题配置文件里有categories: /categories
在编辑站点的source/categories/index.md，添加 

title: categories  
date: 2015-10-20 06:49:50  
type: "categories"  
comments: false

设置标签和分类种类

打开hexo根目录下的配置文件

_config.yml

找到如下位置添加上你所需的分类和标签即可

Category & Tag
default_category: uncategorized
category_map:
    编程: progrmming
       渗透: Web
    life: life
    others: others
    Hexo: hexo
tag_map:
    Java: Java
    漏洞: 漏洞
    XSS: XSS
    SQLi: SQLi
    心得: 心得
    hexo: hexo

编辑新文章时,根据所需填写tags和categories

# hexo
